# Creator AI OS

**Create a week of content in 90 minutes using AI.**

A workflow system for beginner creators, busy professionals, and new personal brands.

> This is not a course. This is not a prompt pack. It is a repeatable system you run every week.

---

## 1. Complete Product Architecture

### The one-sentence promise
You sit down once a week, follow a fixed set of steps with AI doing the heavy lifting, and walk away 90 minutes later with a full week of content ready to post.

### The problem it solves
Most creators don't quit because they lack ideas. They quit because content is unpredictable, unscheduled, and starts from a blank page every single time. Creator AI OS removes the blank page and the guesswork.

### How the system is built (4 layers)

| Layer | What it is | Why it exists |
|-------|------------|---------------|
| **1. The Engine** | A repeatable 5-step weekly workflow | So you never start from zero |
| **2. The Inputs** | Your profile, audience, and content pillars (set up once) | So the AI sounds like *you*, not a robot |
| **3. The Prompts** | Copy-paste prompts for each step of the engine | So you don't have to "know how to prompt" |
| **4. The Home Base** | A Notion workspace that holds everything | So nothing gets lost and you can repeat it forever |

### The Engine: the weekly 90-minute run

This is the heart of the product. Every week, you run the same five steps.

1. **Set the week (10 min)** — Pick your theme and which platform(s) you're posting to.
2. **Generate ideas (15 min)** — Use the Idea prompts to produce 10–15 content ideas from your pillars.
3. **Write the content (35 min)** — Turn the best ideas into captions, hooks, and scripts with the Writing prompts.
4. **Polish and format (20 min)** — Edit for your voice, add CTAs, format per platform.
5. **Schedule and file (10 min)** — Drop everything into your Content Calendar and mark it ready.

That's the whole product in motion. Everything else exists to make these five steps fast and easy.

### Design rules (what we will NOT do)
- No theory. Every page leads to an action.
- No "AI literacy" lessons. Users copy, paste, and tweak.
- No 40-video course. The shortest path to a posted week wins.
- No tool overwhelm. One AI chat tool + one Notion workspace. That's it.

## 2. Deliverables

Everything a buyer receives, grouped by purpose.

### A. Start Here (get a win in 15 minutes)
- **Quick Start Guide (1 page)** — "Do these 3 things now."
- **5-Minute Setup Video / Loom** — Duplicate the Notion template and connect your AI tool.
- **First Win Checklist** — Produce and schedule one post before doing anything else.

### B. The Engine (the core system)
- **The Weekly 90-Minute Workflow** — The 5-step run, written as a checklist you tick off.
- **The Content Calendar** — Notion database where every piece lives (idea → draft → ready → posted).
- **Timer-based run sheet** — Optional time boxes for each step so you stay on track.

### C. The Prompt Library (the "do it for me" part)
Plain, copy-paste prompts. No prompt-engineering knowledge required.
- **Idea prompts** — Generate ideas from your pillars and audience.
- **Hook prompts** — 10 scroll-stopping openers per idea.
- **Caption / post prompts** — Full posts for Instagram, LinkedIn, X, TikTok.
- **Short-form script prompts** — 30–60 second video scripts (Reels, Shorts, TikTok).
- **Repurpose prompts** — Turn one post into 4 formats.
- **Voice prompts** — Make AI write in your tone, not generic AI tone.

### D. The Foundations (set up once, reuse forever)
- **Creator Profile Worksheet** — Who you are, what you post about, who it's for.
- **3 Content Pillars Template** — The 3 topics you'll rotate through.
- **Brand Voice Sheet** — 5 words for your tone + 3 do's and don'ts.
- **Audience Snapshot** — Their problems, in their words.

### E. The Boosters (extra speed and confidence)
- **Swipe File** — 20 fill-in-the-blank post templates.
- **30 Ready-Made Content Ideas** — For weeks you feel stuck.
- **Posting Cheat Sheet** — Best formats per platform, in one page.
- **Troubleshooting Page** — "AI output sounds generic? Do this."

A single duplicatable Notion workspace. Clean, beginner-friendly, no clutter.

### Top-level page tree
```
🏠 Creator AI OS (Home)
│
├── 🚀 Start Here

│     └── This Week's Theme
│
├── 🗓️ Content Calendar   ← main database
│
├── 💬 Prompt Library
│     ├── Idea Prompts
│     ├── Hook Prompts
│     ├── Caption Prompts
│     ├── Script Prompts
│     ├── Repurpose Prompts
│     └── Voice Prompts
│
├── 🧱 My Foundations
│     ├── Creator Profile
│     ├── 3 Content Pillars
│     ├── Brand Voice Sheet
│     └── Audience Snapshot
│
└── 🎁 Boosters
      ├── Swipe File
      ├── 30 Ready-Made Ideas
      ├── Posting Cheat Sheet
      └── Troubleshooting
```

### The Home page (what they see first)
- One-line promise at the top.
- 3 big buttons (links): **Start Here**, **The Weekly Run**, **Content Calendar**.
- "New here? Click Start Here" callout.

### The Content Calendar (the main database)
A single Notion database. These are the columns (properties):

| Property | Type | Example |
|----------|------|---------|
| Title | Text | "5 myths about saving money" |
| Status | Select | Idea / Drafting / Ready / Posted |
| Platform | Multi-select | Instagram, LinkedIn, TikTok |
| Pillar | Select | Pillar 1 / 2 / 3 |
| Format | Select | Caption / Reel / Carousel / Thread |
| Post Date | Date | Mon, Jun 9 |
| Content | Text/Page | The actual caption or script |
| Hook | Text | The opening line |

**Recommended views (built in for the user):**
- **This Week** — filtered to the current week's post dates (Calendar view).
- **By Status** — Board/Kanban: Idea → Drafting → Ready → Posted.
- **Ready to Post** — table filtered to Status = Ready.

### The Weekly Run page (the checklist they live in)
A simple checklist template they copy each week:
```
☐ Step 1 — Set the week (theme + platforms)   ~10 min
☐ Step 2 — Generate 10–15 ideas               ~15 min
☐ Step 3 — Write the content                  ~35 min
☐ Step 4 — Polish + format                    ~20 min
☐ Step 5 — Schedule + file in Calendar        ~10 min
```

### The Prompt Library (how prompts are stored)
Each prompt sits in a toggle so the page stays clean:
```
▸ Idea Prompt (click to open)
   [copy-paste prompt text with [BLANKS] to fill in]
   👉 Paste this into your AI chat tool.
```

### Foundations pages (fill-in-the-blank)
Each is a short page with prompts the user answers once:
- **Creator Profile:** name, niche, what you post about, who it helps.
- **3 Content Pillars:** three rotating topics + 2 example ideas each.
- **Brand Voice Sheet:** 5 tone words, 3 do's, 3 don'ts.
- **Audience Snapshot:** their top 3 problems, in their own words.

### Design principles for the workspace
- **One home, three buttons.** No menu hunting.
- **Toggles over walls of text.** Pages stay short.
- **Copy-paste ready.** Every prompt is one click from the AI tool.
- **Status drives everything.** The whole system runs on the Content Calendar's Status column.
